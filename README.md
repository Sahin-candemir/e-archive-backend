# E-Archive Backend
---

## Genel Bakış
Kullanıcıların dosya yükleme, klasör yönetimi, dosya paylaşımı ve kullanıcı doğrulama işlemlerini RESTful API olarak sunar.  

JWT entegrasyonu sayesinde kullanıcı yetkilendirilir.

---

## Teknolojiler

- Java 17  
- Spring Boot
- Spring Web
- Spring Security (JWT)  
- Spring Data JPA  
- PostgreSQL
- Maven
- Map Struct
